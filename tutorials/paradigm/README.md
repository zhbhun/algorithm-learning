# 范式

- BF 算法：查找/搜索 所有可能性并选择最佳解决方案
- 贪心法：在当前选择最佳选项，不考虑以后情况
- 分治法：将问题分成较小的部分，然后解决这些部分
- 动态规划(Dynamic programming)：使用以前找到的子解决方案构建解决方案
- 回溯法：类似于 BF 算法 试图产生所有可能的解决方案，但每次生成解决方案测试如果它满足所有条件，那么只有继续生成后续解决方案。否则回溯并继续寻找不同路径的解决方案。
